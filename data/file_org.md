# File Structure Overview

```
data/
├── t_set_av1/
├── t_set_av2/
├── trained_models/
├── WOMD_pre_trained_embeddings/
└── AV2_pre_trained_embeddings/

```

## Notes

- **t_set_av1** and **t_set_av2**: Dataset directories, containing raw splits of processed training data
- **trained_models**: Directory containing model checkpoints and weights
- ***_pre_trained_embeddings**: Pre-trained embedding files for initialization or transfer learning
